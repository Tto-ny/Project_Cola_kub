ข้อเสนอโครงการ: ระบบแจ้งเตือนดินถล่มล่วงหน้าจากการคาดการณ์ปริมาณน้ำฝนด้วย ML/DLวิชา: CP413601 การประยุกต์ใช้งานปัญญาประดิษฐ์ในกิจการสาธารณะ
ประยุกต์ใช้เทคนิค ML/DL ในโครงการนี้เป็นการสร้าง "ยามเฝ้าระวังภัยอัจฉริยะ" ที่สามารถเรียนรู้ความสัมพันธ์ที่ซับซ้อนระหว่างปัจจัยด้านภูมิประเทศ ปริมาณฝน ความชื้น และการเกิดดินถล่มในอดีต ซึ่งเกินความสามารถในการวิเคราะห์ของมนุษย์ ทำให้สามารถคาดการณ์ความเสี่ยงได้แม่นยำและล่วงหน้ากว่าการใช้เกณฑ์คงที่ทั่วไป
